# Overview

**wp-module-solutions** handles integration of WordPress Solutions Addon packages (content creator, service, ecommerce) for Bluehost and HostGator customers. It registers with the Newfold Module Loader and uses wp-module-data, wp-module-loader, and wp-module-link-tracker. Maintained by Newfold Labs. Distributed via Newfold Satis.
