# Getting started

Prerequisites: PHP 7.4+, Composer. The module requires wp-module-data, wp-module-loader, wp-module-link-tracker.

```bash
composer install
composer run test
composer run lint
composer run fix
```

See [integration.md](integration.md) for using in a host plugin.
